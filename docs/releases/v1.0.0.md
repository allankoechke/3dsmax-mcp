First stable release of 3dsmax-mcp — MCP bridge for Autodesk 3ds Max **2023–2027**.

## Install

```powershell
git clone https://github.com/cl0nazepamm/3dsmax-mcp.git
cd 3dsmax-mcp
git checkout v1.0.0
uv sync
uv run python install.py
```

Restart 3ds Max after install.

## What's included

- **114 MCP tools** (77 in `core` profile)
- **Prebuilt native plugins** in `native/bin/` for Max 2023–2027 (no SDK required for install)
- **`query_scene`** unified scene reads
- **`smart_import`** — Megascans / folder-aware PBR import
- **`create_shell_material`** — dual render/export material wrapper
- **Tool Inspector** — `Tool Playground.bat`
- **Multi-Max** — MCP Claim This Max
- Bundled **agent skill** + docs ([README](https://github.com/cl0nazepamm/3dsmax-mcp/blob/v1.0.0/README.md), [Advanced](https://github.com/cl0nazepamm/3dsmax-mcp/blob/v1.0.0/docs/ADVANCED.md))

## Breaking changes from 0.8.x

- `create_shell_material` API redesigned (wrap-by-name or texture folder + material classes)
- Scene reads consolidated under `query_scene(action=…)`
- In-Max standalone chat remains **WIP** — use external MCP for production
