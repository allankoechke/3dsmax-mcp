# 1.0.0 ‚Äî First stable release

Production-ready MCP bridge for 3ds Max **2023‚Äì2027**. Prebuilt native plugins ship in the repo ‚Äî `install.py` deploys the matching GUP with no SDK build required. External MCP is the recommended path; in-Max standalone chat remains WIP.

## Scene reads

- Added **`query_scene`** as the unified scene-read surface: `overview`, `filter`, `class`, `property`, `selection`, `delta`.
- Removed legacy snapshot modules; use `query_scene` or `get_session_context` on demand instead of session preambles.
- Added **`get_instances`** and **`get_dependencies`** for instancing and reference-graph reads.

## Import & materials

- Added **`smart_import`** ‚Äî folder-aware mesh + PBR import for Megascans-style assets, shared atlases, and multi-variant plant bundles (`Var1_LOD0` ‚Ä¶ `Var8_LOD0` in one folder).
- Reworked **`create_shell_material`** ‚Äî wraps any two scene materials in `Shell_Material`, or builds from `texture_folder` with `render_material_class` / `export_material_class` (OpenPBR, Physical, Arnold, Octane, etc.). Not Arnold/UberBitmap-only anymore.
- Fixed **`scatter_forest_pack`** ‚Äî per-geometry `widthlist` / `heightlist` footprints so all Forest Pack variants scatter, not just the first.
- **`palette_laydown`**: added `random_per_subfolder` sampling for GSG/Megascans-style libraries.

## Native bridge & tooling

- Prebuilt **`mcp_bridge_2023.gup` ‚Ä¶ `mcp_bridge_2027.gup`** in `native/bin/` (version-stamped **1.0.0**).
- Named pipe is default transport; **MCP Claim This Max** routes clients when multiple Max instances are open.
- Added **Tool Inspector** ‚Äî `Tool Playground.bat` for manual one-tool-at-a-time testing with full tripback.
- Added live smoke harness: in-Max **MCP Smoke** macro, `run_tool_smoke`, and `scripts/run_live_tool_smoke.py`.
- **`create_object`** / **`clone_objects`**: ground-contact placement (`pos_mode="ground"`) with spatial tripback (`bbox`, `placement`, `groundContact`).

## Skill and docs

- Bumped package/native version to **`1.0.0`**.
- Trimmed agent skill to **3ds Max usage only** ‚Äî no repo/build/native-dev content.
- Rewrote **README** (user-facing install + full tool list); moved architecture/build/chat internals to **`docs/ADVANCED.md`**.
- Added **`CHANGELOG.md`** and release checklist updates.

## Breaking changes

- **`create_shell_material`** ‚Äî new parameters; old `render_material_name` / `base_color_path` / `orm_path` API removed.
- Scene reads ‚Äî use **`query_scene(action=‚Ä¶)`** instead of removed snapshot tools.
- Standalone in-Max chat ‚Äî experimental; prefer external MCP for production.

## Validation

- `native\build.bat all` ‚Äî GUPs built for Max 2023‚Äì2027
- `uv run python -m pytest tests/ -q` ‚Äî unit tests (material shell, smart import, query scene, etc.)
- Live Max: shell wrap, smart import variants, Forest Pack scatter verified on Max 2027
